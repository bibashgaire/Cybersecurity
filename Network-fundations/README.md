***A network is a group of devices connected together to share data. Example home wifi, school network, office network, the internet etc. Mainly it is divided into two parts LAN (Local Area Networl) and WAN (Wide Area Network)***

## LAN
It is a small area network like home, school, office. Eg. my laptop and printer connected to the same Wi-Fi.

## WAN
It is a large area network eg. internet


## MAC Address 
Physical address assigned to the network card. like 00:1A:2B:3C:4D:5E, usally its fixed or doesnt changed.

## IP Address
Logical address used for communication, in more simple term home address of all electronic devices connected to a network. Eg 192.168.1.10

## Private IP Address 
Used inside a LAN like in home or office, not directly accessible from the internet.
range

10.0.0.0 - 10.255.255.255

172.16.0.0 - 172.31.255.255

192.168.0.0 - 192.168.255.255

## Public IP Address 
It is assigned by our ISP, which is visible to the internet. Everyone in a LAN may share the same public IP.

## Packets
Data travels in small pieces called packets. Large files are broken into packets and reassembled at the destination.

## DNS 
Translate the domain name into the ip address like www.google.com to 142.25.x.x, without dns it will be very difficult to memorize all the numerical value.

## TCP vs UDP (Delivery System)
TCP is reliable
- Checks delivery
- Resends lost packets
- Slower but safe
  used in website, login pages, emails etc.
UDP is fast chaos
- No guarantee packet arrive
- No waiting
- Just speed
  used in gaming, live streams, calls

## Ports (Service Doors)
A single IP can run many services. Ports seperate them.
Examples 

443 -- HTTPS(secure web)

80 -- HTTP(normal web)

22 -- SSH(remote login)
