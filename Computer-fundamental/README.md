***Computer fundamentals are the basic principles and skills required to understand how computer work, operate and interact with data. This foundational knowledge covers the core components of computing systems, including hardware (such as the CPU, memory, and input/output devices), software (operating systems and applications), data processing (the input-process-output cycle), networking basics, and cybersecurity principles.***

**Hardware:** *Hardware refers to the physical components of a computer or a machine that we can see and touch. It contains circuit, board ICs, or other electronic in a computer system. Eg. Motherboard, Monitor, Keyboard etc.*
  
  1.CPU(Central Processing Unit): CPU is the primary hardware component of a computer also in easy language its a brain of a computer. CPU processes instructions to make your computer do what you want. Without a CPU, a         computer would just be a lifeless box of parts.

  2.RAM(Random Access Memory): It serves as a computer’s short-term, volatile memory for active tasks. It temporarily stores data and instructions that the CPU needs immediate access to, allowing for fast loading of            applications, multitasking, and smooth performance.

  3.Storage Device: A storage device is a hardware component used to store, retrieve, and manage digital data such as documents, photos, software, and operating systems eg. Hard Disk Drives (HDDs), Solid State Drives           (SSDs), and USB flash drives.

  4.Motherboard: A motherboard is the main printed circuit board (PCB) in a computer system that acts as the central hub, connecting all internal components like the CPU, RAM, and storage devices.

  5.Input/Output Devices: Input and output devices are hardware components that facilitate communication between a computer system and the external environment, such as human users or other systems eg. Monitor, keyboard        etc.

**Software:** *Software is a collection of porocedures, instructions, documentation that tells a computer exactly what to do or allows users to interact with a computer.*
