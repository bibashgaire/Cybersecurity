***An Operation System(OS) is system software that acts as bridge in between computer hardware and its user. It manages critical system resources such as the CPU, memory, storage, and input/output devices, ensuring they are allocated efficiently and securely among competing processes. Eg.Windows, Linux, macOS***


**Kernal**
*Kernal is the core of the OS it control CPU, memory, Hardware etc.*

**User Space**
*It is where programs run like chrome, spotify etc. Programs ask the kernal to interact with hardware.*

**Process and Threads**
*Process is a running program each process has its own memory eg. Chrome, Termina etc. A smaller execution unit inside a process is thread one process can have many threads eg. chrome { thread 1, thread2}. Threads share memory and make multitasking faster.*

***Linux Filesystem***
